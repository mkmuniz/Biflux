# Setup
`node >= 18.x.x` & `npm >= 6.x.x`

Create a `.env` file into `front` folder and fill out these informations:

```env
NEXTAUTH_URL=
NEXTAUTH_SECRET=

AWS_IAM_USER_SECRET_KEY=
AWS_IAM_USER_ACCESS_KEY=
```

# How To Run

### Dev environment

- `npm install`
- `npm run dev`

### Dev environment

- `npm run build`
- `npm start`

Access it `http://localhost:3000`
